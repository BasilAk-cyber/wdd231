Personal Movie Library

Site Plan & Design Document

Site Name

CineVault

Rationale: The name "CineVault" combines "cinema" with "vault," suggesting a personal treasure trove of movies. It conveys the idea of carefully curating and storing your favorite films and watchlist in a secure, organized digital space. The name is memorable, modern, and immediately communicates the site's purpose as a movie collection and tracking tool.

Optional Domain: cinevault.app or cinevault.io (these domains reflect the modern web app nature of the project)

Site Purpose

CineVault serves as a comprehensive personal movie discovery and tracking platform that empowers users to:

  • Discover New Movies: Browse a dynamic catalog of popular and trending films fetched from The Movie Database (TMDB) API with detailed information including ratings, release dates, and synopses.
  • Build a Watchlist: Save movies to a personal watchlist that persists across sessions using browser local storage, allowing users to track films they want to see.
  • Search and Filter: Find specific movies quickly using search functionality and filtering options to narrow down the catalog.
  • View Detailed Information: Access comprehensive movie details through an interactive modal interface, including cast, crew, runtime, genres, and user ratings.
  • Recommend Movies: Submit movie recommendations through a contact form, fostering a community-oriented approach to movie discovery.

The site provides a seamless, client-side experience without requiring user accounts, making it immediately accessible and privacy-friendly while demonstrating modern web development practices including API integration, asynchronous JavaScript, local data persistence, and responsive design.

Scenarios

These scenarios represent typical questions and use cases from the target audience:

Scenario 1: Discovering New Content

"I'm bored and want to find something good to watch tonight. What are some highly-rated movies I haven't seen yet?"

How CineVault Addresses This: The home page displays a curated selection of popular movies with ratings and posters. Users can browse the Movies Catalog page to explore 20+ films with detailed information, filtering by rating or genre, and clicking on any movie card to view full details in a modal before adding it to their watchlist.

Scenario 2: Managing a Watchlist

"I keep forgetting which movies my friends recommended. How can I keep track of what I want to watch?"

How CineVault Addresses This: Users can click the "Add to Watchlist" button on any movie card or modal. The watchlist is prominently displayed on the home page and persists across browser sessions using local storage. Users can easily remove movies once they've watched them.

Scenario 3: Finding Specific Movies

"I heard about a specific movie released in 2023, but I can't remember the full title. How can I find it?"

How CineVault Addresses This: The search and filter functionality on the home page allows users to search by title keywords or filter by release year, helping them quickly locate specific movies from the API database.

Scenario 4: Sharing Recommendations

"I just watched an amazing independent film that's not in the database. How can I suggest it be added?"

How CineVault Addresses This: The About/Contact page features a "Recommend a Movie" form where users can submit their name, email, and movie suggestion. The form data is displayed on a confirmation page, acknowledging their contribution.

Accessibility Note: All color combinations have been selected to meet WCAG AA contrast standards (4.5:1 for normal text, 3:1 for large text). The crimson accent on dark backgrounds provides excellent visibility while the off-white text ensures readability.

Typography

CineVault uses a two-font system that balances bold, cinematic impact with clean readability:

Bebas Neue

Font Family: 'Bebas Neue', sans-serif

Usage: Main headings (h1, h2), page titles, logo, section headers, featured movie titles

Characteristics: Bold, condensed, all-caps friendly. This display font creates strong visual hierarchy and evokes classic movie poster typography with its commanding presence and cinematic feel.

DM Sans (400, 500, 700)

Font Family: 'DM Sans', sans-serif

Usage: Body text, paragraphs, movie descriptions, navigation links, form labels, buttons (when not using Bebas Neue), subheadings (h3, h4)

Characteristics: Clean, modern sans-serif with excellent readability at all sizes. The geometric letterforms maintain clarity in both light and dark themes. Multiple weights (400 regular, 500 medium, 700 bold) provide flexibility for emphasis and visual hierarchy within body content.

Implementation: Both fonts are loaded via Google Fonts with font-display: swap for optimal performance. Bebas Neue provides cinematic branding while DM Sans ensures comfortable reading of movie descriptions and content.
